# 「沉淀」计划

尝试在滚滚向前的世界之中沉淀出自己的一片空间，会随缘地慢慢更新一些我的笔记

刚刚转移到 rspress，还有很多东西没有整理。

制作一个静态的 **个人主页** / **交大门户主页** / **个人主页**
要求：

- 页面具有 `header`、`sidebar`、`content`、`footer` 等基本结构

- content 的内同形式至少5种（文本，标题，图片，列表，表格）

- 使用 flex 布局,使用至少5个提到的 felx 属性

- 使用到至少两种 position

- 页面中要有一个悬浮组件，里面包含文本和图片

- 响应 pc 和 mobile，两种页面有不同布局

- 使用 2 种伪类

- 使用两种伪元素

- 编码习惯优雅

- 使用 css 动画加分

- 使用 css transform 加分

- 可以在网上寻找灵感，但不可 copy 抄袭
